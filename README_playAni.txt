# playAni.py Animation player

## Overview
This is a Python script that demonstrates the features of the multilineMAX7219 library by creating a dynamic animation using MAX7219/8x8LED arrays. The script reads sprite files from a specified folder and displays them as an animated sequence.

## Requirements
- Python 3.x
- multilineMAX7219 library

## Installation
1. Ensure you have Python 3.x installed on your system.
2. Install the multilineMAX7219 library by following the instructions in its repository: [multilineMAX7219](https://github.com/FMMT666/multiline_MAX7219).

## Usage
1. Run the script with the following command:
   ```bash
   python3 playAni.py <folder_name>
